* About
Very simple Lisp interpreter in Rust.

* Installation
#+begin_src console
$ git clone https://github.com/shoumodip/lust
$ cd lust
$ cargo install --path .
#+end_src

* Usage
** Repl
Launch a REPL, use =CTRL-d= or =CTRL-C= to exit.

#+begin_src console
$ lust
#+end_src

** Run files
Run files sequentially, exits in case of error.

#+begin_src console
$ lust file1 [file2] [file3]
#+end_src
